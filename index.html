Center Table Schedule Simulator

Description: (click to expand)

Day Selector (Upper Left Corner):
       Select the day you want to run the simulator for.

Clock (Below Day Selctor):
       Shows the vitural time inside the simulator.

Buttons (Below):
      Start:
            Start the simulation. For the simulator to run correctly, please do not switch to other programs or web tabs while the simulator is running.
      Stop:
            Stop the simulation. The simulation can not be resumed once stopped.
      Clear:
            Clear everything on the page ans refresh the page.

Schedule Selector (Below Buttons):
      Center Table Schedule:
            Use the schedule created by Center Table management team for Autumn, 2018 to run the simulation.
      Model Generated Schedule:
            Use the schedule generated by our model to run the simulation.

Mode Selector (Below Buttons):
      Monte Carlo Simulation:
            Distribute customers randomly based on the probability of a customer decides to eat at any station. Result for this mode differs between each run.
            Fixed Probability: Distribute fixed number of curtomers to all station at any time base on probability. Result for this mode remains the same for each run.

Speed Selector (Above Simulation Area):
       Select the speed of the simulation by letting different amount of time in real world to represent 15 minutes in the simulator.

Result Area (Upper Right Corner):
      Total Customers Count:
            Records the total number of customers arrived so far.
      Customers Waiting Time:
            Records the sum of waiting time of LAST customers every 15 minutes.
